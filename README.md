# Mechanic Shop Website

A responsive website for a car repair shop built with HTML, CSS, and JavaScript.

## Features

- Loading screen with animated wrench and key icons
- Hero video section with professional car repair footage
- About Us section
- Services section
- Contact form
- Footer with social media icons

## File Structure

```
src/
├── index.html          # Main HTML file
├── styles.css          # CSS styles
├── script.js           # JavaScript functionality
└── assets/
    ├── README.md       # Instructions for media files
    ├── poster-video.mp4 # Hero video (placeholder)
    └── about-image.jpg  # About section image (placeholder)
```

## Setup

1. Clone or download the project files
2. Open `index.html` in a web browser
3. Replace placeholder media files in the `assets/` folder with actual images and videos

## Customization

- Edit `index.html` to change content
- Modify `styles.css` for styling changes
- Update `script.js` for additional functionality

## Dependencies

- Font Awesome (loaded via CDN) for social icons

## Browser Support

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Responsive design for mobile and desktop
